## Anti-patterns (사용 금지)
- 장식용 무한 애니메이션 (로딩 인디케이터만 허용)
- linear easing (UI 트랜지션에는 ease-out/ease-in 사용)
- alt 없는 img, label 없는 input
- 시각적으로만 에러 표시 (aria-live 또는 role="alert" 필수)
- 4.5:1 미만 텍스트 대비 (다크 모드 7:1 권장)
- 컴포넌트 내 하드코딩 컬러 (CSS 변수/Tailwind 토큰 사용)
- 순수 검정(#000000) 배경 — slate-900(#0F172A) 사용
- Arial / Roboto 기본 폰트
- 카드 안에 카드 중첩
- prefers-reduced-motion 미적용
- 3D 차트 효과 (2D 데이터에 왜곡 발생)
- 과도한 그라디언트 (데이터 가독성 저하)

## Stack-specific rules
- React 19 + TypeScript strict mode
- Tailwind CSS 4 (no tailwind.config — @theme in CSS)
- Recharts for histogram/bar/timeseries, @nivo/heatmap for correlation
- Zustand for state — 컴포넌트에서 useStore 직접 접근
- react-window for virtualized data table
- 인라인 style 금지 (Recharts 내부 제외)
- Props는 interface로 명시
