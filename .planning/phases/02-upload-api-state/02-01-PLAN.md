    ```python
    class SummaryStats(BaseModel):
        mean: float | None = None
        std: float | None = None
        min: float | None = None
        max: float | None = None
        q1: float | None = None
        median: float | None = None
        q3: float | None = None
        skewness: float | None = None

    class CorrelationMatrix(BaseModel):
        columns: list[str]
        values: list[list[float | None]]

    class OutlierResult(BaseModel):
        column: str
        lower_bound: float
        upper_bound: float
        outlier_count: int
        outlier_indices: list[int]

    class QualityAlert(BaseModel):
        column: str
        alert_type: str   # "constant" | "high_cardinality" | "high_null" | "high_skew"
        message: str
        severity: str     # "warning" | "info"

    class AnalysisResultResponse(BaseModel):
        summary: dict[str, SummaryStats]
        correlation: CorrelationMatrix
        outliers: list[OutlierResult]
        quality_alerts: list[QualityAlert]
        row_count: int
        column_count: int
        missing_ratio: float
        duplicate_count: int
    ```

    **2. Create `backend/app/services/__init__.py`** (empty file).

    **3. Create `backend/app/services/parser_service.py`:**
    - `parse_file(contents: bytes, filename: str) -> tuple[pd.DataFrame, list[ColumnMeta]]`
    - Route by filename extension: `.csv` -> `pd.read_csv(io.BytesIO(contents), low_memory=False)`, `.json` -> `pd.read_json(io.BytesIO(contents))`
    - After parsing, call `df.infer_objects()` for secondary type inference (ANLZ-04)
    - `detect_column_type(series: pd.Series) -> ColumnType`:
      - `pd.api.types.is_numeric_dtype(series)` -> `numeric`
      - `pd.api.types.is_datetime64_any_dtype(series)` -> `datetime`
      - Try `pd.to_datetime(series, format='mixed')` in a try/except -> `datetime` if succeeds
      - `series.nunique() / max(len(series), 1) < 0.5` -> `categorical`
      - Otherwise -> `text`
    - Build `ColumnMeta` list from detected types, `series.isnull().any()` for nullable, `series.nunique()` for unique_count
    - Raise `ValueError` for unsupported extensions, empty DataFrames (0 rows)

    **4. Create `backend/app/services/stats_service.py`:**
    - `analyze(df: pd.DataFrame) -> dict[str, SummaryStats]`
    - For each numeric column: compute mean, std, min, max, Q1 (0.25 quantile), median, Q3 (0.75 quantile), skewness (using `scipy.stats.skew`)
    - Handle `pd.NA` / `NaN` by converting to `None` before creating `SummaryStats`
    - Skip non-numeric columns

    **5. Create `backend/app/services/correlation_service.py`:**
    - `analyze(df: pd.DataFrame) -> CorrelationMatrix`
    - `df.select_dtypes(include='number').corr(method='pearson')`
    - Convert to `CorrelationMatrix(columns=list, values=nested_list)`
    - Replace `NaN` with `None` in the values matrix
    - If fewer than 2 numeric columns, return empty matrix (`columns=[], values=[]`)

    **6. Create `backend/app/services/outlier_service.py`:**
    - `analyze(df: pd.DataFrame) -> list[OutlierResult]`
    - For each numeric column: compute Q1, Q3, IQR = Q3 - Q1
    - Bounds: lower = Q1 - 1.5 * IQR, upper = Q3 + 1.5 * IQR
    - Outlier indices: `df.index[mask].tolist()` where mask = `(col < lower) | (col > upper)`
    - Return `OutlierResult` per column

    **7. Create `backend/app/services/quality_service.py`:**
    - `analyze(df: pd.DataFrame) -> list[QualityAlert]`
    - Constant column: `series.nunique() <= 1` -> warning
    - High cardinality: categorical-typed column where `nunique / len > 0.9` -> info
    - High null ratio: `series.isnull().sum() / len > 0.5` -> warning
    - High skew: numeric column where `abs(skewness) > 2` -> info
    - Return list of `QualityAlert`

    **8. Create test fixtures:**

    - `backend/tests/fixtures/sample.json`: The same data as JSON array.

    - Test `parser_service.parse_file` with empty content: expect `ValueError`
    - Test `parser_service.detect_column_type` for each type
    - Test `stats_service.analyze`: verify mean/median match known values from fixture
    - Test `correlation_service.analyze`: verify matrix is symmetric, diagonal is 1.0

    **1. Create `backend/app/api/upload.py`:**
    - `router = APIRouter()`
    - `POST /api/upload` accepting `file: UploadFile`:
      - Read file contents: `contents = await file.read()`
      - Size check: `if len(contents) > 10 * 1024 * 1024: raise HTTPException(status_code=413, detail="File size exceeds 10MB limit")`
      - Extension check: extract from `file.filename`, reject if not `.csv` or `.json` with HTTP 400
      - Call `parser_service.parse_file(contents, file.filename)` to get `(df, columns)`
      - Store df in module-level dict `_datasets: dict[str, pd.DataFrame] = {}` keyed by a generated `file_id` (use `uuid.uuid4().hex[:8]`)
      - Return `UploadResponse(columns=columns, preview=df.head(50).to_dict(orient='records'), row_count=len(df))` with `file_id` in a custom header `X-File-Id` (so analyze knows which dataset to use)
      - NOTE: Also add `file_id` to the response body. Update `UploadResponse` in `backend/schemas/upload.py` to include `file_id: str` field.

    **2. Create `backend/app/api/analyze.py`:**
    - `router = APIRouter()`
    - `POST /api/analyze` accepting JSON body `{"file_id": str}`:
      - Look up df from `upload._datasets[file_id]`; if not found, raise HTTP 404 "Dataset not found. Please upload a file first."
      - Run all 4 services in `run_in_executor` to avoid blocking the event loop (ANLZ-03):
        ```python
        loop = asyncio.get_event_loop()
        executor = ThreadPoolExecutor(max_workers=2)
        stats = await loop.run_in_executor(executor, stats_service.analyze, df)
        correlation = await loop.run_in_executor(executor, correlation_service.analyze, df)
        outliers = await loop.run_in_executor(executor, outlier_service.analyze, df)
        quality = await loop.run_in_executor(executor, quality_service.analyze, df)
        ```
      - Compute dataset-level metrics: `row_count`, `column_count`, `missing_ratio` (total nulls / total cells), `duplicate_count` (`df.duplicated().sum()`)
      - Return `AnalysisResultResponse(summary=stats, correlation=correlation, outliers=outliers, quality_alerts=quality, row_count=..., column_count=..., missing_ratio=..., duplicate_count=...)`

    **3. Update `backend/app/main.py`:**

    - Add `from app.api.analyze import router as analyze_router`
    - Add `app.include_router(upload_router)` and `app.include_router(analyze_router)` after the health router
