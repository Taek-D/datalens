class SummaryStats(BaseModel):
    mean: float | None; std: float | None; min: float | None; max: float | None
    q1: float | None; median: float | None; q3: float | None; skewness: float | None

    **1. Create `frontend/src/hooks/useUpload.ts`:**
    - Returns: `{ isDragging, error, uploadProgress, handleDragOver, handleDragLeave, handleDrop, handleFileSelect, resetUpload }`
    - Internal state: `isDragging` (boolean), `uploadProgress` (number 0-100)
    - `validateFile(file: File): string | null` — checks extension (.csv, .json only) and size (10MB max). Returns error message string or null.
    - `handleDragOver(e: React.DragEvent)` — `e.preventDefault()`, set isDragging true
    - `handleDragLeave(e: React.DragEvent)` — set isDragging false
    - `handleDrop(e: React.DragEvent)` — `e.preventDefault()`, set isDragging false, extract `e.dataTransfer.files[0]`, validate, upload
    - `handleFileSelect(e: React.ChangeEvent<HTMLInputElement>)` — extract `e.target.files?.[0]`, validate, upload
    - Internal `processUpload(file: File)`:
      - Call `useStore.getState().setStatus('uploading')` and `useStore.getState().setError(null)`
      - Call `uploadFile(file)` from analysisApi
      - On success: call `setDataset(response.preview, response.columns, response.row_count, response.file_id)`, then set status to 'analyzing' (analysis hook takes over)
      - On error: set status to 'error', set error message (extract from axios error response or use generic message)
    - `resetUpload()` — calls `useStore.getState().resetStore()`

    **2. Create `frontend/src/hooks/useUpload.test.ts`:**
